2. 모듈 설명
    1. 공통 구성
        1. 슬라이드 분할
            1. DB에서 내용 불러온 후 한 PPT당 몇 줄씩 표기할 건지 선택
            2. 아래 항목 조절 가능
                1. 2줄
                2. 4줄
                3. 직접 입력
                    1. 내용 전체 편집 가능한 텍스트창 생성됨
                    2. Enter 두 번 기준으로 새 슬라이드 생성 후 내용 분할됨
        2. 폰트
            1. 토글로 접고 펴기 가능
            2. 아래 항목 조절 가능
                1. 폰트
                    1. 드롭다운 형식으로 선택 가능
                2. 크기
                    1. 숫자 입력 및 up down 버튼 클릭으로 조절 가능
                3. 굵기
                    1. 바(게이지)로 조절 가능
                4. 색상
                    1. 검정, 흰색, 무지개 (커스텀) 조절 가능
                5. 글자 정렬
                    1. 왼쪽 정렬, 가운데 정렬, 오른쪽 정렬 선택 가능
        3. 배경
            1. 토글로 접고 펴기 가능
            2. 아래 항목 조절 가능
                1. 단색 배경
                    1. 단색 배경색 선택 가능
                2. 템플릿 배경
                    1. 저장된 템플릿 배경색 선택 가능
                3. 이미지 배경
                    1. 이미지 업로드를 통해 배경색 선택 가능
    2. 찬양
        1. 설명 : DB에 저장된 찬양곡 선택하여 PPT 생성
        2. 모듈 이름 : 곡명
        3. 구성
            1. 곡 검색
                - 검색창에 곡명 입력 시, DB에서 일치하는 곡 리스트 표시하여 곡명 선택
                - 곡 없을 경우 검색 결과가 없습니다. 출력됨
            2. 슬라이드 분할
            3. 폰트
            4. 배경
    3. 찬송가
        1. 설명 : DB에 저장된 찬송가 선택하여 PPT 생성 (이미지로 불러옴)
        2. 모듈 이름 : 곡명
        3. 구성
            1. 곡 검색
                - 검색창에 곡명 입력 시, DB에서 일치하는 곡 리스트 표시하여 곡명 선택
                - 곡 없을 경우 검색 결과가 없습니다. 출력됨
    4. 성경
        1. 설명 : DB에 저장된 성경 정보에 따라 선택하여 PPT 생성
        2. 모듈 이름 : 성경이름 장:시작 절-끝 절
        3. 구성
            1. 성경 선택
                1. 구약, 신약 분류
                2. 번역본 : 드롭다운으로 설정
                3. 성경 : 드롭다운으로 설정
                4. 장 : 드롭다운으로 설정
                5. 시작 절-끝 절 : 텍스트 입력
                    1. 유효 범위 텍스트로 출력
                    2. 유효 범위에 해당되지 않는 범위 입력 시 아래 오류 문구를 미리보기 및 작업 영역에 출력
                        - DB에 있는 절 범위는 n-n절입니다.
            2. 슬라이드 분할
            3. 폰트
            4. 배경
    5. 설교
        1. 설명 : 금주 설교 제목 및 시리즈 표기
        2. 모듈 이름 : 설교 제목
        3. 구성
            1. 설교 제목 : (로직 기입 필요)
            2. 제목 폰트
            3. 설교 시리즈 : (로직 기입 필요)
            4. 시리즈 폰트
            5. 배경
    6. 광고
        1. 설명 : 교회 소식 및 광고 내용 슬라이드 생성
        2. 모듈 이름 : 교회 소식 (고정값)
        3. 구성
            1. 광고 내용 구성
                1. 내용별로 토글 버튼 생성
                2. 삭제 버튼
                3. 광고 내용
                    1. 숫자 옆 텍스트 작성
                4. 내용 폰트
                5. 광고 세부 내용
                    1. 광고 내용 옆 세부 내용 작성
                6. 세부 폰트
                7. 박스 배경색
                    1. 광고 내용 박스 배경색 선택
                    2. 검은색, 흰색 선택 가능
            2. 광고 추가 버튼
                1. 광고 내용 4개 이상 추가 시, 3개 항목씩 슬라이드 분할되어 생성됨
            3. 배경
        4. 특이사항

        2. 모듈 이름 : 제목
        3. 구성

                        2. 제목 : (로직 기입 필요)
